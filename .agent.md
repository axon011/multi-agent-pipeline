# .agent.md - multi-agent-pipeline

## Project Info

Multi-agent system that automatically researches topics and generates structured reports using LangGraph state machines and CrewAI role definitions.

## Key Technologies
- LangGraph (state machine orchestration)
- CrewAI (agent roles)
- FastAPI (REST API)
- OpenAI GPT-4o

## Directory Structure

```
multi-agent-pipeline/
├── app/
│   ├── main.py              # FastAPI entrypoint
│   ├── agents/
│   │   ├── planner.py       # Planning agent
│   │   ├── researcher.py    # Research agent
│   │   └── writer.py        # Writing agent
│   ├── graph/
│   │   └── pipeline.py     # LangGraph pipeline
│   ├── routes/
│   │   └── research.py      # /research endpoint
│   └── models/
│       └── schemas.py       # Pydantic models
├── tests/
│   └── test_pipeline.py
├── docker-compose.yml
├── Dockerfile
└── requirements.txt
```

## Key Endpoints
- `POST /research` - Generate research report

## How to Identify This Project
- **Keywords**: LangGraph, CrewAI, research pipeline, multi-agent
- **Unique feature**: Uses state machine for agent orchestration
- **NOT**: This is NOT llmops-dashboard (that's React/monitoring) or rag-eval-system (that's RAG/Qdrant)
